##  Project Overview

- The metro network is modeled as a **graph** using an adjacency matrix.
- Each station represents a **node**.
- Each direct connection represents an **edge**.
- The system computes:

  -  Shortest path (minimum stations)  
  -  Minimum interchange route  
  -  Estimated travel time  
  -  Number of interchanges
